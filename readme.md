Askbot Vagrant Development environment
======================================

Contact: Marton Kiss <marton.kiss@gmail.com>

Prerequisites:
- Virtualbox (Latest)
- Vagrant (>=1.5.1)

Launch box:

    $ git clone https://github.com/mkissam/vagrant-askbot-dev.git
    $ cd vagrant-askbot-dev
    $ vagrant up
    $ vargant ssh
